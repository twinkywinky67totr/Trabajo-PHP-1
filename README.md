# Trabajo‑PHP‑1

Proyecto desarrollado en PHP para la gestión de usuarios y subida de fotos.  

## Descripción

Web diseñada por Jose Conde para:

- Registrar y autenticar usuarios.
- Subir y gestionar fotos.
- Listar y editar información de usuarios.
- Aplicar buenas prácticas en PHP, HTML, CSS y manejo de sesiones.

## Estructura del repositorio

- `assets/` : recursos estáticos (CSS, JavaScript, imágenes decorativas).  
- `config/` : archivos de configuración, como parámetros de la base de datos.  
- `includes/` : librerías o archivos PHP reutilizables.  
- `public/` : archivos accesibles públicamente (index.php, login.php, register.php, etc.).  
- `uploads/photos/` : carpeta para almacenar imágenes subidas por los usuarios.  

## Tecnologías y requisitos

- Lenguaje: PHP 8.x o superior  
- Servidor web: Apache en Xampp
- Control de versiones: Git
